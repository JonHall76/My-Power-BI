# My-Power-BI
📊 Power BI Retail & Sales Analytics Dashboard
A Data Technician Bootcamp Project
📝 Overview
This project was completed as part of a Data Technician bootcamp, focusing on transforming raw retail and sales data into an interactive, insight‑driven Power BI dashboard.
The goal was to apply industry‑standard BI practices—data cleaning, modelling, DAX, and visual design—to tell a clear and compelling data story.

🔧 Key Skills Demonstrated
🧹 Data Transformation & Cleaning (Power Query)

Imported data from multiple sources (Excel spreadsheets, OData feeds).
Cleaned and prepared datasets using Power Query:

Removed errors and duplicates
Standardised column formats
Performed table merging and appending
Created calculated columns during shaping


Built repeatable, documented transformation steps as part of the query pipeline.

📐 Data Modelling & DAX

Created meaningful relationships between fact and dimension tables.
Added DAX calculated columns and measures, including:

Sales totals and comparisons
Year‑to‑date and month‑to‑date metrics
Profitability indicators


Implemented dynamic calculations used across visuals and filters.

📊 Interactive Reporting & Visualisation
Built a multi‑page report featuring a variety of Power BI visuals:

Bar charts for product and category performance
Line charts for trend analysis
Pie and donut charts for sales composition
Maps to highlight regional activity
Tables & matrices for granular breakdowns
Slicers and filters to enable user‑driven exploration of the data

🎨 Data Storytelling
The dashboard was designed to highlight key retail insights, such as:

Top‑selling products and categories
Revenue trends over time
Best‑performing locations
Customer behaviour patterns
Operational KPIs supporting business decisions

The final report tells a cohesive story about business performance while remaining intuitive and visually engaging.

🚀 What I Learned
Throughout this project, I gained hands‑on experience with:

Transforming messy datasets into reliable analytical models
Applying DAX to create dynamic, reusable calculations
Designing business‑ready dashboards
Using filters, slicers, and drill‑downs for interactivity
Structuring data stories for non‑technical audiences
